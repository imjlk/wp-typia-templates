# Compound Patterns

A parent-and-child WordPress block scaffold with InnerBlocks, optional persistence wiring, and hidden implementation child blocks

## Template

compound

## Development

```bash
bun install
bun run start
```

## Build

```bash
bun run build
```

## Type Sync

```bash
bun run sync-types
```

`src/types.ts` remains the source of truth for `block.json` and `typia.manifest.json`.
