# `create-wp-typia`

Compatibility shim for [`@wp-typia/create`](https://www.npmjs.com/package/@wp-typia/create).

Use this package when you need historical commands such as:

- `bun create wp-typia`
- `npx create-wp-typia`

For new installations, prefer:

- `npx @wp-typia/create`
- `bunx @wp-typia/create`
